## 🎯 两个视图的区别

### Bookmark Tree (原有视图)
- 纯粹的书签树展示
- 显示所有书签和文件夹
- 支持展开/折叠
- 支持搜索和过滤
- 有撤销功能
- **没有 Canvas 画布功能**

### Bookmark Canvas (新增视图)
- Canvas 画布工作区（点阵背景）
- 中心永久栏目（显示书签树）
- 可以拖拽创建临时节点
- 可以在画布上移动节点
- 可以拖回永久栏目（添加到浏览器书签）
- 支持导入 HTML/JSON 书签
- 支持导出 .canvas 文件
- **专门用于可视化规划和整理**

## 🔄 使用建议

### 使用 Bookmark Tree 当你想：
- 快速浏览所有书签
- 搜索特定书签
- 简单的查看和点击

### 使用 Bookmark Canvas 当你想：
- 规划书签结构
- 整理混乱的书签
- 临时收集和比较书签
- 导入外部书签并筛选
- 可视化地管理书签
