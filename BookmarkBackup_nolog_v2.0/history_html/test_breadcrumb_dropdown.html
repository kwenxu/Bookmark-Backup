面包屑下拉菜单测试

将鼠标悬停在按钮上，会显示向下的箭头图标。点击箭头图标可以打开下拉菜单选择候选项。

/

测试说明：

  1. 观察按钮右侧有预留的空白位置（下拉图标的位置）
  2. 将鼠标悬停在"2024年"或"11月"按钮上
  3. 会看到预留位置显示出向下的箭头图标
  4. 点击箭头图标会打开下拉菜单
  5. 菜单会直接定位到当前选中的项目（无滚动动画）
  6. 点击候选项会更新选中项并关闭菜单

功能特点：

  • ✅ 左右空白对称且紧凑（各20px）
  • ✅ 下拉图标始终占据空间，UI不会跳动
  • ✅ 默认透明不可见，悬停时显示
  • ✅ 打开菜单时直接定位到当前选中项
  • ✅ 无滚动动画，立即显示
  • ✅ 选中项居中显示
