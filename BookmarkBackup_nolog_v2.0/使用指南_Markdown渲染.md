# Markdown 渲染使用指南

## 快速开始

### 1. 在空白栏目中使用

**步骤：**
1. 打开「书签画布」（Bookmark Canvas）
2. 在空白处双击，创建一个空白栏目
3. 再次双击栏目进入编辑模式
4. 输入你的 Markdown 文本
5. 按 `Ctrl+Enter`（Mac: `Cmd+Enter`）保存
6. 文本会自动渲染为格式化内容

**示例：**
```markdown
### 项目进度

1、<font color="#ffff00">正在实现</font>的功能
2、<font color="#00b0f0">还没有做</font>的功能
3、<u>已完成</u>的功能

> [!tip] 提示
> 双击可以编辑内容
```

### 2. 在临时栏目说明中使用

**步骤：**
1. 创建或选择一个临时栏目
2. 找到栏目顶部的说明区域
3. 点击编辑按钮（铅笔图标）
4. 输入 Markdown 文本
5. 点击外部区域或按快捷键保存

说明区的语法与渲染规则与「空白栏目」完全一致（支持 Markdown + 安全 HTML 白名单，如 `<u>`、`<font color="">`、`<center>`、`<p align="">` 等）。

### 3. 在永久栏目说明中使用

**步骤：**
1. 进入「书签画布」视图
2. 在「书签树（永久栏目）」顶部的说明区域双击
3. 输入 Markdown 文本
4. 点击外部区域保存

说明区的语法与渲染规则与「空白栏目」完全一致（支持 Markdown + 安全 HTML 白名单，如 `<u>`、`<font color="">`、`<center>`、`<p align="">` 等）。

## 常用语法速查

### 文字颜色
```markdown
<font color="#ffff00">黄色</font>
<font color="#00b0f0">蓝色</font>
<font color="#ff0000">红色</font>
<font color="#00ff00">绿色</font>
```

### 文字样式
```markdown
**粗体**
*斜体*
<u>下划线</u>
~~删除线~~
==高亮==
```

### 对齐（与空白栏目格式工具一致）
```markdown
<center>居中</center>
<p align="left">左对齐</p>
<p align="center">居中</p>
<p align="right">右对齐</p>
```

### 提示框（Callout）
```markdown
> [!note] 笔记
> 这是笔记内容

> [!tip] 提示
> 这是提示内容

> [!warning] 警告
> 这是警告内容
```

### 列表
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

- [ ] 未完成任务
- [x] 已完成任务
```

## 常见问题

### Q: 为什么我的 HTML 标签显示为源码？

**A:** 可能的原因：
1. 标签没有正确闭合（如缺少 `</font>`）
2. 使用了不支持的标签
3. 需要刷新页面才能看到效果

**解决方法：**
- 检查标签是否正确闭合
- 参考支持的标签列表
- 刷新页面（F5）

### Q: 支持哪些颜色格式？

**A:** 只支持十六进制颜色格式：
- ✅ `#ffff00`（6位）
- ✅ `#ff0`（3位简写）
- ❌ `yellow`（颜色名称不支持）
- ❌ `rgb(255,255,0)`（RGB格式不支持）

### Q: 可以嵌套使用标签吗？

**A:** 可以，但要注意正确的嵌套顺序：
```markdown
<font color="#ff0000"><u>红色下划线</u></font>
**<font color="#00ff00">绿色粗体</font>**
```

### Q: Callout 不显示怎么办？

**A:** 确保格式正确：
- 必须以 `>` 开头
- `[!type]` 中的类型名称要正确
- 每行都要以 `>` 开头

正确示例：
```markdown
> [!note] 标题
> 内容第一行
> 内容第二行
```
