# 连接线工具栏快速参考

## 工具栏按钮（从左到右）

| 图标 | 功能 | 说明 |
|------|------|------|
| 🗑️ | 删除 | 删除选中的连接线 |
| 🎨 | 颜色 | 6种预设颜色 + 自定义RGB |
| 🔍+ | 定位 | 自动定位并放大到连接线 |
| ↔️ | 方向 | 切换连接线的起点和终点 |
| ✏️ | 标签 | 添加或编辑连接线文字标签 |

## 使用流程

1. **选择连接线** → 点击任意连接线
2. **工具栏显示** → 自动显示在点击位置附近
3. **执行操作** → 点击工具栏按钮
4. **取消选择** → 点击画布空白区域

## 颜色预设

| 编号 | 颜色 | 十六进制 |
|------|------|----------|
| 1 | 红色 | #ff6666 |
| 2 | 橙色 | #ffaa66 |
| 3 | 黄色 | #ffdd66 |
| 4 | 绿色 | #66dd99 |
| 5 | 蓝色 | #66bbff |
| 6 | 紫色 | #bb99ff |

## 快捷提示

- **标签编辑**: 输入空文字可删除标签
- **方向切换**: 可多次点击来回切换
- **颜色保存**: 所有设置自动保存
- **主题适配**: 标签自动适配浅色/深色主题
- **点击标签**: 点击标签文字也可选中连接线
